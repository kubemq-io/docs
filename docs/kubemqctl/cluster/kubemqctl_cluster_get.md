### Get Command

Get information of KubeMQ of clusters command

### Synopsis

Get command allows to show the current information of KubeMQ of clusters

```
kubemqctl cluster get [flags]
```

### Examples

```

	# Get status of KubeMQ of clusters
	kubemqctl cluster get

```

### Options

```
  -h, --help   help for get
```
