### Delete Command

Delete KubeMQ cluster command

### Synopsis

Delete command allows deleting one or more KubeMQ clusters deployments

```
kubemqctl cluster delete [flags]
```

### Examples

```

 	# Delete KubeMQ cluster
	kubemqctl cluster delete

```

### Options

```
  -h, --help   help for delete
```
